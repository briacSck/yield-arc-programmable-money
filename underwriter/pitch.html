The Underwriter

Insurance for an autonomous CFO agent.

A bounded agent is a measurable agent — and a measurable agent is an insurable agent.

bounded→insurable→scalable

YIELD is an autonomous CFO agent that manages idle USDC treasury for small businesses — trading live on Circle's Arc testnet under an on-chain mandate contract that hard-bounds it (balance floor, per-move ticket, daily cap) with no human in the loop. An agent that moves real money is hard to trust and impossible to insure — and uninsurable means unscalable. The Underwriter is the answer: an independent Claude agent that, every day, assesses that CFO at arm's length from public read-only data, verifies it stayed inside its bounds, and prices the risk.

How each run works

01

Fetch

Reads the CFO's live public dashboard — stats, decisions, mandate snapshot.

02

Cross-check

Writes its own viem script to read the mandate contract on-chain and reconcile it.

03

Price

Computes risk stats and a 30-day premium from a fully disclosed formula.

04

Certify

Issues a certificate + memo, and remembers the run so it can report the trend.
